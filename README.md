# Toyxona Booking System

This project is a wedding hall booking system built with React.js.

## Features
- View wedding halls
- Select available dates
- Booking form
- Admin panel
- Owner panel

## Technologies Used
- React.js
- React Router
- Tailwind CSS
- JavaScript

## Pages
Home page – shows wedding halls  
Hall detail – booking system  
Admin panel – manage halls  
Owner panel – manage bookings
